trajminer.clustering.DBSCAN¶

class trajminer.clustering.DBSCAN(eps=0.5, min_samples=5, measure=None, n_jobs=1)¶

DBSCAN Clustering.

eps : float (default=0.5)
The maximum distance between two trajectories for them to be considered in the same neighborhood.
min_samples : int (default=5)
The minibum number of trajectories in a neighborhood for a trajectory to be considered as a core point, including the trajectory itself.
measure : SimilarityMeasure object (default=None)
The similarity measure to use for computing similarities. See trajminer.similarity.
n_jobs : int (default=1)
The number of parallel jobs.

Ester, M., Kriegel, H. P., Sander, J., & Xu, X. (1996, August). A density- based algorithm for discovering clusters in large spatial databases with noise. In Kdd (Vol. 96, No. 34, pp. 226-231).
